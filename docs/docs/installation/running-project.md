# Running the Viewer Locally

Once your data is in place, start the React application to explore the archive.

```bash
cd instagram-archive-viewer
npm install       # first time only
npm run dev
```

Open `http://localhost:5173` (or the port shown in the terminal) in your browser. Use the
search bar and conversation list to navigate messages.

- You can toggle perspective with the button in the top-right of the UI.
- Media such as photos and videos appear inline when available.
