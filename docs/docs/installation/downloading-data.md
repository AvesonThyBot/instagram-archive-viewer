# Downloading Your Instagram Data

This guide explains how to request and retrieve a JSON export of your Instagram messages.

## Steps

1. Open **Instagram Settings** → **Your Activity** → **Download Your Information**.
2. Choose **Messages** and set the format to **JSON**. HTML exports are not compatible.
3. Pick a date range (or "All of Your Data") and submit the request.
4. When the archive email arrives, download and unzip the folder. The `messages` directory is what you'll use.

## Debugging & Troubleshooting

> _(add notes here about common issues, e.g. missing messages.json, request taking too long, expired links, etc.)_

- **No data received yet?** Make sure you selected the correct date range and waited for the email from Instagram.
- **JSON file not found** – verify the archive structure; the `messages` folder may be inside a nested ZIP.
